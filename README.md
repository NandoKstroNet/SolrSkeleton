SolrSkeleton
====================

Esqueleto usado na apresentação de solr. Seja livre pra baixar, brincar e mandar um feedback. 

### Requirements 
Java 6    
Jetty (server)    

### Como usar essa treta ? 

`git clone`    
`cd solrSkeleton`   
`./solr.sh`    


o arquivo "commands" pode te ajudar :)
